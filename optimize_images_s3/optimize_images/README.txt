Este script optimiza las imágenes JPG y PNG para reducir su tamaño sin perder calidad.

REQUERIMIENTOS:
- /bin/bash
- jpegtran (librería de optimización para jpeg)
- optipng (librería de optimización para png)

EJECUCIÓN:
$ ./optimize.sh {ruta absoluta al directorio}

Por ejemplo:
./optimize.sh /home/user/site1/images
